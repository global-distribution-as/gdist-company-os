# SOP: Onboarde ny leverandør

**Mål:** Fra første kontakt til produkter live på plattformen — strukturert og uten hull.

---

## Fase 1 — Første kontakt

- [ ] Avtal intro-møte (video eller e-post)
- [ ] Innhent: firma-info, produktkatalog, MOQ, lead times, betalingsbetingelser
- [ ] Opprett leverandør-notat i `02_Suppliers/` med status: `Prospekt`

## Fase 2 — Evaluering (Daniel godkjenner)

- [ ] Sjekk: kvalitetssertifikater, referanser, prisbilde
- [ ] Vurder: passer produktene våre buyer-base?
- [ ] Beslutning: **Gå videre** eller **Avslå** — skriv begrunnelse i notatet

## Fase 3 — Avtale

- [ ] Send NDA (mal: `_Templates/nda-supplier.md`)
- [ ] Send leverandøravtale (Daniel signerer)
- [ ] Bekreft: prisliste, rabattnivåer, eksklusivitet (hvis aktuelt)
- [ ] Oppdater status: `Avtaleinngått`

## Fase 4 — Produktoppsett

- [ ] Motta produktdata: bilder, beskrivelser, EAN/SKU, vekt/mål
- [ ] Last opp produkter til plattformen (se: `09_Tech/produktimport.md`)
- [ ] Kvalitetsssjekk: bilder OK? Beskrivelser OK? Pris riktig?
- [ ] Oppdater status: `Aktiv`

## Fase 5 — Aktivering

- [ ] Send velkomst-e-post til leverandør med kontaktinfo og rutiner
- [ ] Legg til i leverandør-oversikten i `02_Suppliers/README.md`
- [ ] Informer Martin/Jessica om nye produkter som er live

---

**Viktige regler**
- Ingen produkter live uten signert avtale
- Alltid dobbeltsjekk MOQ og lead time mot hva vi sier til buyers
- Produktbilder må være hvit bakgrunn eller tydelig produktfoto

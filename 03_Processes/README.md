# Prosesser – Global Distribution AS

Hvordan selskapet fungerer i grove trekk.

## Kjerneflyt

```
Leverandør (EU) → Global Distribution AS → Kjøper (Asia)
```

1. **Leverandør** laster opp produkter og priser via Supplier Portal
2. **Admin** setter marginer og godkjenner katalog
3. **Kjøper** blar i katalog, sender forespørsel / ordre via Buyer Portal
4. **Leverandør** mottar og fullfører ordren

# Admin

Internal operations portal for Global Distribution AS staff.

## Portal

Route: `/admin/*`
Accent: Purple

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Dashboard | `/admin/dashboard` | Platform overview |
| Suppliers | `/admin/suppliers` | Manage suppliers |
| Buyers | `/admin/buyers` | Manage buyers |
| Products | `/admin/products` | Product catalogue |
| Inventory | `/admin/inventory` | Inventory management |
| Orders | `/admin/orders` | All orders |
| Settings | `/admin/settings` | Platform settings |

## Capabilities

- Manage supplier and buyer accounts
- Set pricing with margins
- Oversee inventory
- Full order visibility across both domains
